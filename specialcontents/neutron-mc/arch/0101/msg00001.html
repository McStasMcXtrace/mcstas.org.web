Dear fellow McStas Users,

we have recently developed a small add-on for McStas:

StatMon is an add-on package for McStas developed at the
Max-Planck-Institut für Metallforschung, Stuttgart. 

McStas includes a number of monitors/detectors. These all work fine,
with the one restriction that their output is only visible at the end of
a simulation. 
StatMon supplies a number of slightly modified monitor components (along
with other software) which allow the user to constantly control the
status of a simulation while it is running, hereby discovering errors in
the instrument definition, before hours of calculation time might be
wasted. 
StatMon allows monitoring of multiple processes on one single or on
multiple machines distributed over the network. 

At the moment only a beta version is available, which still may contain
bugs and/or annoyances.
It can be obtained from  http://129.187.179.18
