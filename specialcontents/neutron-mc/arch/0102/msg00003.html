One new - but minor - issue came up here in Garching today. The maximum
number of neutrons which can be processed in a single run is probably
limited (I didn't try it out though for obvious reasons :-) to something
like 2**31-1. Now I am getting to a point where I might need more...

neutrons than the number 'intmax'.

(I like to avoid having to merge the output of several seperate runs
manually)
