Dear Tain-Shang Chang,

It sounds like you have successfully installed McStas on your
computers, which is good. You do not say which version of McStas you are
using (the command "mcstas --version" will tell you). I would recommend
that you use the latest release, version 1.2. You should also make sure
that you have the latest manual (I can send a hardcopy if you have
trouble downloading and printing).

If McStas has been fully installed, you can use the "mcplot" and "mcgui"
programs to plot the results. See the manual sections 2.6.1 and
2.6.5. Otherwise you need some other program (gnuplot, matlab, origin,
excel, ...). The data files are in ascii formats that most programs will
read, see the manual section 2.7.

For a simple introduction to the use of the porgram you should read the
McStas tutorial. This may be downloaded from

    http://neutron.risoe.dk/mcstas/tutorial/tutorial.pdf

I am not sure if anyone have attempted detailed SANS simulations using
McStas, but there should be no fundamental problems.

 - Kristian.
