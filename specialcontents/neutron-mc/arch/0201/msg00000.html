Hello,

In the framework of the EU funded SCANS project we have setup a search
engine covering all the MC packages involved in this project.
With this engine one can find all information in one place. 

For the moment it covers Mcstas (Risoe+ILL), vitess, nexus, restrax, and
rmc websites. If you think that more websites should be included, please
let me know.
The engine can be found at http://irisfm.iri.tudelft.nl/scans

We hope that this engine will be of any use to you.
