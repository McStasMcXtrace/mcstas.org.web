Hy Kristian,

I've updated the ILL web site, to point to your official components page, and to my reduced unofficial web page.
You will see there the Monitor_nD, which replaces all other existing monitors easely.

For instance (I just put the option string)
"energy" is equivalent to E_monitor
"x y" is equivalent to PSD_monitor
"theta phi" is a PSD_4PI
"list all neutrons kx ky kz energy" is a dump of all passing neutrons (k,w)
you can also use 4 shapes for detector, as well as many options.
