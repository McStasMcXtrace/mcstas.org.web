Well, that is difficult to say without more information. But the most
likely cause is a problem with the PGPLOT library and the pgperl
interface between this library and Perl.

Could you try running the following one-line command:

  perl -e 'use PGPLOT;pgbeg(0,"/xserv",1,1);pgenv(0,1,0,2,0,"");pgend'

and send me the output? This is a small Perl program that uses PGPLOT;
if it works it should produce a window with an empty graph in it.

Also information on the machine and operating system you were using
would be helpful.
