This looks very much like a problem with gcc on the Silicon Graphics
Workstation. You can try to build McStas with the standard C compiler
"cc" and see if it works any better:

    rm config.cache
    setenv CC cc
    ./configure
    make
    make install

In any case, let me know what pops up, and I will try to help if needed.
