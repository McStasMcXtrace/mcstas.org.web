The neutron-mc mailing list

NOTE: These pages are OBSOLETE, and only provided to make up for possible broken links on other pages.

Please use the new mailman interface for access to the mailing list.

To create a forum for the discussion of issues related to Monte Carlo simulation of neutron scattering, the "neutron-mc" mailing list has been set up. To post a message to the list, send an e-mail to neutron-mc@risoe.dk.

The possible subjects of e-mails posted to this list include

  • Announcements of new software releases, results, papers, etc.
  • Discussion of simulation techniques for neutron scattering.
  • Discussion on the use of particular simulation programs.
  • Anything else related to Monte Carlo simulation of neutron scattering.
In case of problems, contact Kristian Nielsen <kristian.nielsen@risoe.dk>
