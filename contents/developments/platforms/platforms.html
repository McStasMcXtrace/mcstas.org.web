System requirements for McStas

McStas has been tested on various systems (see extensive tests further). All of them were found to run McStas, eventhough some of the tools (GUI) could fail.

The complete installation for McStas requires:

  1. a C compiler
  2. perl and perl-Tk
  3. Matlab OR (pgplot+pgperl+PDL) OR Gnuplot (Scilab 4.0 is also supported but not recommended)
  4. Optional stuff: VRML viewer, MPI, octave+gnuplot

WARNING:

McStas 1.12a ONLY works with scilab 4.0 (All systems).

We recommend Ubuntu and Debian systems, easy to install and provide all the necessary software. Matlab runs on these system as well. Microsoft Windows installer is now available for a very easy installation from a complete package (C compiler, Scilab, VRML viewer, McStas, perl)

The color codes indicate:

- McStas builds + works fully (including gui + at least one plotting backend)
- McStas builds + works (no gui and/or no plotting backend)
- McStas does not build (no mcstas)
