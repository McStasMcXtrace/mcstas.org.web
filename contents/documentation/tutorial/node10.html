Exercise: Insert a sample

  1. Now, insert a V_sample() component after the last PSD_monitor and L_monitor and a Beamstop component 1.5 m after the sample. The V_sample() component simulates a Vanadium sample. Such a sample scatters incoherently, i.e. in all directions. At the same position as the sample, insert a PSD_monitor_4PI component of radius 1.0 m. Read the documentation for details on input parameters. Run a simulation. Notice the number of hits.
