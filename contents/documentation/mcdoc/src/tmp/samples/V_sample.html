The V_sample component

Vanadium sample.

Identification

  • Author: Kim Lefmann and Kristian Nielsen
  • Origin: McStas 1.5
  • Date: 15.4.98
  • Version: 1.6

Description

A Double-cylinder shaped incoherent scatterer (a V-sample)
No multiple scattering. Absorbtion included.

Example: V_sample(radius_i=0.001,radius_o=0.01,h=0.02,focus_r=0.035,pack=1, 
          target_x = 0, target_y = 0, target_z = 0.235)

Input parameters

Parameters in boldface are required; the others are optional.
Name Unit Description Default
radius_i m Inner radius of sample in (x,z) plane  
radius_o m Outer radius of sample in (x,z) plane  
h m Height of sample y direction  
focus_r m Radius of sphere containing target. 0.035
pack 1 Packing factor 1
target_x 0
target_y m position of target to focus at 0
target_z 0.235

Output parameters

None.
