The Monitor_4PI component

Monitor that detects ALL non-absorbed neutrons. Example: Monitor_4PI()

Identification

  • Author: Kim Lefmann and Kristian Nielsen
  • Origin: McStas 1.5
  • Date: April 17, 1998
  • Version: 1.5

Description

Counts ALL neutrons that propagate this far in the instrument, regardless
of origin or direction. Mostly used for test purposes.
