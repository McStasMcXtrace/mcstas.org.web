The PSD_monitor_4PI component

Spherical position-sensitive detector.

Identification

  • Author: KL and KN
  • Origin: McStas release
  • Date: April 17, 1998

Description

An (n times m) pixel spherical PSD monitor using a cylindrical projection. 
Mostly for test and debugging purposes.
