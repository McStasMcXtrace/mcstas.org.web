The PSD_monitor component

Position-sensitive monitor.

Identification

  • Author: KL
  • Origin: McStas release
  • Date: Feb 3, 1998
  • Version: 1.7

Description

An (n times m) pixel PSD monitor. This component may also be used as a beam
detector.
