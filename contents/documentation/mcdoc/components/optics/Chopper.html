The Chopper component

Disk chopper.

Identification

  • Author: Philipp Bernhardt
  • Origin: McStas release
  • Date: Januar 22 1999
  • Version: 1.3

Description

Models a disc chopper with n identical slits, which are symmetrically disposed on the disc.

Example values: w=0.05 R=0.5 f=2500 n=3 pha=0

Input parameters

Parameters in boldface are required; the others are optional.
Name Unit Description Default
w m Width of the slits at the bottom side  
R m Radius of the disc  
f rad/s angular frequency of the Chopper (algebraic sign defines the direction of rotation  
n 1 Number of slits  
pha s Phase
