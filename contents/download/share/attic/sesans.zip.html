Simulation package for simulating a SESANS instrument

A package of files needed to run the sesans.instr Instrument with McStas 1.12. This is a simulation file created to simulate the SESANS-instrument at Delft IRI, The Netherlands. This SESANS variant uses spin-flip foils inside precssion magnets.

Created by E. B. Knudsen, Risø DTU with great help from Linda Udby, Niels Bohr Institute.
