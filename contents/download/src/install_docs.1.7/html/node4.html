Windows build

  • Start by unpacking the mcstas-1.7-src.tar.gz package using e.g. Winzip.
  • Using an ansi-c compiler (we recommend Bloodshed Dev-C++ - easy to install and use (Section 6.1)), the McStas package can be compiled using the build.bat script of the mcstas-1.7 directory you just unpacked. Follow the on screen instructions.
  • When the build has been done (e.g. mcstas.exe has been produced), proceed to install (Section 5).
