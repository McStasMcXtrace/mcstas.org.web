Support packages for Win32 systems

The following packages have been tested and are known to work on Windows XP with the current release of McStas (1.12b plus dev release 20101101).

  • Install the Perl 5.10 package - use all defaults
  • Install the Dev-Cpp package (gcc compiler
  • Add the 
    bin
    directory of the Dev-Cpp install to the Windows PATH
  • Install the Scilab package
  • Install the Cortona VRML
  • Add the 
    bin
    directory of the Scilab install to the Windows PATH
