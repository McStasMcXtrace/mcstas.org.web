Test results for the source and monitor/detector components

This experiment tests the functioning of the Source_flat component and the various monitors/detectors. The instrument setup consists of the source aimed at a flat PSD monitor, with a 4PI PSD monitor, a 4pi single monitor, a plain single monitor, and a energy-sensitive monitor overlapped on the flat PSD.

See also the source code for the instrument definition.

This is the output of the 4pi PSD monitor that shows how the source aims all neutrons towards the target.

This is the output of the flat PSD. This PSD was placed at the source target and was made somewhat bigger than the target area. The slightly fussy edges of the target area are believed to be artifacts of small-angle approximations in the source calculations.
