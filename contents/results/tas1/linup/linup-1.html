Monochromator line-up: PHM scan

Measurements

Open collimation, 20 meV neutrons. 10.5x40.5mm slit on sample position with monitor placed 20mm behind. Scan PHM from -3 to 3 for 5 sec/point. Data file linup010.dat.

Simulations

2TM is set to 74° as determined from drawings of the TAS1 instrument. The monochromator mosaicity is specified to 30' but not precisely known. Simulations for different values of the mosaicity was performed as follows:

Mosaicity Peak width
40 0.446
45 0.489
50 0.513
55 0.527
60 0.546

Based on this, we continue the simulations with 45' mosaicity on monochromator and analysator.

This is the result linup_1_45.sim of simulation with 45' mosaicity. Instrument definition linup-1.instr. 

  Amplitude     3.6271e-01    2.3164e-03
  Centre       -3.7077e+01    1.8214e-03
  Width        -4.8897e-01    1.9908e-03
  Background   -1.3489e-03    3.8135e-04

We observe that the peak is 0.077° off-center, reflecting an asymetry in the beam channel from the cold source to the monochromator. In the following simulations, PHM is set to -37.077°.
